интерфейс из 2х вкладок
Списки пациентов должны загружаются сразу для 2-х вкладок(Присутствует и Выбывшие) асинхронно.
Загрузка данных осуществляется с помощью ajax запросов к приложенным json-файлам. 
После завершения загрузки данных в заголовке вкладок нужно выводится число загруженных пациентов для соответствующей вкладки(например Присутствует(27))
При клике на пациенте в таблице в левую панель подгружается дополнительная информация о выбранном пациенте
